# 自注意力

一种**同一序列内部**的注意力机制——Query、Key、Value 均来自同一个输入序列。让序列中的每个位置都能直接关注到序列中的任意其他位置。

## 与普通注意力的区别

| 类型 | Q 来源 | K/V 来源 |
|---|---|---|
| 普通注意力 | 外部模块 | 外部模块 |
| **自注意力** | **同一序列** | **同一序列** |

## 自注意力的直观理解

以句子 "The cat sat on the mat because it was tired" 为例：

- 翻译"it"时，自注意力可以直接找到它指代"cat"（通过高注意力权重）
- 这个连接不依赖线性距离，即使"it"和"cat"相隔很远也能直接建模

传统 RNN 需要经过多步传递才能建立这个连接（距离越远，信号衰减越多）。

## 自注意力在 Transformer 中的位置

```
编码器层：
  输入 → [自注意力层] → LayerNorm → [前馈网络] → 输出

解码器层：
  输入 → [掩码自注意力层] → LayerNorm
               ↓
        [交叉注意力层]（Q 来自掩码注意力，K/V 来自编码器）
               ↓
        [前馈网络] → 输出
```

## 计算复杂度

自注意力的计算复杂度为 **O(n² · d)**（n = 序列长度，d = 模型维度）。这是 Transformer 的主要计算瓶颈，也是后续改进（如 FlashAttention、Sparse Attention）的优化目标。

# 多头注意力

将注意力机制**并行运行多次（多个"头"）**，让模型能够同时关注来自不同表示子空间的信息。

## 核心思想

单次注意力只能学习一种"查询-键-值"关系。多头注意力让模型能够**同时从多个角度关注信息**：

```
MultiHead(Q, K, V) = Concat(head_1, ..., head_h) · W^O
where head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)
```

| 符号 | 含义 |
|---|---|
| h | 注意力头数量（论文中使用 h=8） |
| d_k = d_model/h | 每个头的维度（512/8 = 64） |
| W_i^Q, W_i^K, W_i^V | 每个头的投影矩阵（可学习） |
| W^O | 最终输出投影矩阵（可学习） |

## 为什么有效

不同的注意力头倾向于学习不同类型的关系：
- 某些头擅长捕捉**句法依赖**（主语→动词）
- 某些头擅长捕捉**语义相似性**（同义词）
- 某些头负责**指代消解**（代词→实体）

这使得模型能够并行建模多种语言现象。

## 论文中的配置

| 参数 | Base 模型 | Big 模型 |
|---|---|---|
| 注意力头数 h | 8 | 16 |
| d_model | 512 | 1024 |
| 每个头维度 d_k | 64 | 64 |
| 前馈维度 | 2048 | 4096 |

## 消融实验结论

原论文消融实验表明：
- 多头注意力比单头注意力效果好
- h=16 略优于 h=8 和 h=4
- d_k=64（配合 d_model=1024 时）是效果与效率的最佳平衡

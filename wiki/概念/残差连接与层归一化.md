# 残差连接与层归一化

Transformer 中每层子层周围的两个关键稳定机制，使极深网络（6+ 层堆叠）能够有效训练。

## 残差连接（Residual Connection）

```
Output = x + Sublayer(x)
```

将子层的输入直接加到输出上。如果子层学习的是恒等映射（对任务无帮助），梯度可以无衰减地反向传播到更底层。

**核心价值**：解决了深层网络的梯度消失问题，使 6 层甚至上百层的堆叠成为可能。

## 层归一化（Layer Normalization）

```
LayerNorm(x) = γ · (x - μ) / √(σ² + ε) + β
```

- μ：均值，σ²：方差
- γ、β：可学习的缩放和偏移参数
- ε：数值稳定项（10⁻⁶）

对每个样本独立归一化（与 BatchNorm 不同），适合 RNN 和小 batch 场景。

## 在 Transformer 中的组合

两者组合在一起，构成了 Transformer 的标准子层结构：

```
x → Self-Attention → + x → LayerNorm → FFN → + norm(x) → LayerNorm → 输出
```

消融实验表明：**同时移除残差连接和层归一化会导致训练崩溃**，两者缺一不可。

## 与其他归一化方法的比较

| 方法 | 归一化维度 | 适合场景 |
|---|---|---|
| BatchNorm | batch 维度 | 图像分类（CNN）、大 batch |
| LayerNorm | 特征维度 | NLP、RNN、小 batch |
| RMSNorm | 特征维度（仅均值） | 效率优先的场景（更少计算） |
| DeepNorm | 残差后按比例缩放 | 超深层 Transformer |

# 数据摄入

将原始材料转换为 LLM 可处理的格式的过程，是 [[Wiki 编译]] 的上游步骤。

## 常见来源

| 来源类型 | 处理方式 |
|---|---|
| 网页文章 | Obsidian Web Clipper 插件 → `.md` |
| 图片 | 下载到本地，与 `.md` 同目录，供 LLM 直接引用 |
| 论文 / PDF | 提取文本转为 `.md` |
| 代码仓库 | 克隆到本地，提取关键文件转为 `.md` 摘要 |
| 数据集 | 提取 README 和文档为 `.md` |

## 本地图片原则

摄入网页文章时，将图片下载到本地至关重要——这样 LLM 可以直接引用（无需在查询时抓取 URL）。这对于承载语义的图表、数据可视化尤为重要。

## raw/ 目录

所有摄入的来源统一放入 `raw/` 目录。LLM **不编辑** raw/——它是不可变的原始来源。编译后的 wiki 单独存放在 `wiki/`。

# BestBlogs & TrendRadar — 摘要

**来源：** [[@Jason23818126]](https://x.com/Jason23818126/status/2039903486046068938) | 2026-03-28

## Summary

本文介绍了三个用于自动化信息聚合的 GitHub 开源工具，帮助用户构建高价值信息流：BestBlogs 聚合 400+ 优质信源并用大模型进行质量打分和摘要提取；TrendRadar 一键监控知乎、B站、微博、抖音等 35 个平台的热搜，支持关键词过滤和 Webhook 推送；CloudFlare-AI-Insight-Daily 基于 Cloudflare Workers 每日自动生成 AI/科技领域简报，支持网页、播客脚本、RSS、微信公众号等多形态输出。三者共同构成了一个完整的信息获取-监控-分发 pipeline。

## Key Points

- **BestBlogs**：聚合 X、播客、博客 400+ 信源；大模型提纯质量打分、生成一句话摘要；外文自动翻译；播客文字稿提取并区分发言人
- **TrendRadar**：35 个主流平台热搜监控；自定义关键词过滤（如"AI、开源"）重新排序；无需代码，配置 Webhook 即可推送到企微/钉钉/飞书
- **CloudFlare-AI-Insight-Daily**：运行于 Cloudflare Workers（免费额度可用）；高垂直度抓取模型/GitHub 热门/学术论文/科技大V动态；多形态输出：网页日报 / 播客脚本 / RSS / 微信公众号
- 核心价值：绕过各平台 App 的信息噪音，让 AI 过滤提纯后的高密度资讯主动推送

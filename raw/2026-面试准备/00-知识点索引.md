## 面试策略速查

| 策略 | 题目 |
|------|------|
| **100% 必问** | AI-LangGraph Q1-Q2（状态机/Attack Stage）、RAG Q5（混合检索） |
| **简历提到就必追** | Token 成本优化、技术债量化、30% 效率提升、上手周期缩短 |
| **容易出彩加分** | DSL 设计（代码+可视化）、Prompt 版本化管理、Bad Case 分析 |
| **诚实承认更可信** | Recall 监控缺失、A/B 测试数据不完整 |
| **避免抱怨** | 离职原因（腾讯云→新机会） |
| **数据敏感度** | 每个数字都可能追问测量方法（500+ 任务、2周→3天） |
| **防守必问** | 最大技术挑战、最自豪项目、失败经历 |
| **反向提问** | 问团队技术栈、培养机制 |
| **安全业务必问** | 告警定级标准、误报率控制、等保合规 |
| **反侦察识别** | 50+ 剧本谁写的？LangGraph 重头写的还是改的？ |
